:::note Roadmap — not yet available
Webhooks are **not available** in the current release. This page describes a planned capability and is published for reference only. The REST API does not currently expose any webhook endpoints. Until webhooks ship, poll the [REST API](https://api.numeraire.tech/docs) to retrieve updates.
:::

Webhooks are a way for an app to provide other applications with real-time information. A webhook delivers data to other applications as it happens, meaning you get data immediately.
Unlike typical APIs where you would need to poll for data very frequently in order to get it real-time. This makes webhooks much more efficient for both provider and consumer.

When this feature becomes available, it will be documented here with the supported events and payloads.
